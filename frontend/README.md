# Frontend - OSEAN POS

This is the frontend application for the OSEAN POS system, built with Next.js and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
npm start
```

### Linting
```bash
npm run lint
```

### Deployment
Refer to the [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying).

---

## Features
- Responsive design
- State management with Zustand
- Optimized for performance and accessibility
